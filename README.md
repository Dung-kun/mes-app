# Template Catra Mobile

Flutter mobile application scaffold for the Laravel + Vue migration.

## Current status

- Clean architecture folder structure
- Riverpod app shell
- Go Router navigation
- Dio client with interceptor hooks
- Mock repositories for initial features
- Material 3 theme foundation

## Pending bootstrap work

- Run `flutter create .` inside `mobile_app` once the local Flutter CLI is responding normally to generate platform folders.
- Run `flutter pub get`
- Run `dart run build_runner build --delete-conflicting-outputs`
